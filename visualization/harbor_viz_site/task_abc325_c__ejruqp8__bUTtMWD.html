Problem Statement

There are zero or more sensors placed on a grid of H rows and W columns. Let (i, j) denote the square in the i-th row from the top and the j-th column from the left. Whether each square contains a sensor is given by the strings S_1, S_2, ‚Ä¶, S_H, each of length W. (i, j) contains a sensor if and only if the j-th character of S_i is #. These sensors interact with other sensors in the squares horizontally, vertically, or diagonally adjacent to them and operate as one sensor. Here, a cell (x, y) and a cell (x', y') are said to be horizontally, vertically, or diagonally adjacent if and only if \max(|x-x'|,|y-y'|) = 1. Note that if sensor A interacts with sensor B and sensor A interacts with sensor C, then sensor B and sensor C also interact. Considering the interacting sensors as one sensor, find the number of sensors on this grid. Input The input is given from Standard Input in the following format: H W S_1 S_2 \vdots S_H Output Print the answer. Model Solution

def solve():
    H, W = map(int, input().split())
    grid = []
    for _ in range(H):
        grid.append(input().strip())

    # Track visited cells
    visited = [[False] * W for _ in range(H)]

    # DFS to explore connected components
    def dfs(i, j):
        if i < 0 or i >= H or j < 0 or j >= W:
            return
        if visited[i][j] or grid[i][j] == '.':
            return

        visited[i][j] = True

        # Check all 8 adjacent cells (horizontal, vertical, diagonal)
        for di in [-1, 0, 1]:
            for dj in [-1, 0, 1]:
                if di == 0 and dj == 0:
                    continue
                dfs(i + di, j + dj)

    # Count connected components
    count = 0
    for i in range(H):
        for j in range(W):
            if grid[i][j] == '#' and not visited[i][j]:
                dfs(i, j)
                count += 1

    print(count)
